JavaScript

JavaScript - это интерпретируемый язык программирования общего назначения с динамической типизацией данных.

Язык программирования общего назначения означает. что на этом языке можно решать практически любые задачи по программированию.

Интерпретируемый означает, что программы, написаные на этом языке выполняются в исходном виде, т.е., исходный кот НЕ преобразуется в машинный или промежуточный код. Выполняется именно исходный код. Интерпретатор - это программа которая выполняет другие программы. Недостатком интерпретируемых языков является низкая производительность. Преимуществом интерпретируемых языков является высокая кросплатформенность. Программа будет работать везде, где работает интерпретатор. Это дает возможность использовать JavaScript для разработки серверных приложений как под Windows так и пол Unix, разработки мобильных приложений для iPhone/Android и т.д. Но несмотря на все эти возможности, наибольшее распространение JavaScript получил именно в Web-разработке, а именно для разработки Front-End части Web-приложений. Хотя на сегодняшний день также используются и для разработки Back-End части. Front-End - Client-side, Back-End - Server-side. Следовательно самым распространенным интерпретатором языка JavaScript является браузер.

То что программа никуда не компилируется, а выполняется сам исходный код этой программы, говорит о том, что нет ошибок ни компиляции и компоновки, а есть только ошибки выполнения, что часто усложняет поиск ошибок.

Динамическая типизация данных означает то, что у переменной нет конкретного типа, её тип зависит от того, какое в ней хранится значение и при смене значения также может сменится и тип переменной. .... перед 02:00 Полиморфизм (многоформенность) - это способность объектов вести себя по разному в зависимости от обстоятельств (статический полиморфизм), или же способность объектов вести себя по разному в зависимости от того, кем они являются (динамический полиморфизм). Статический полиморфизм реализуется перегрузкой функций, а динамически при помощи указателей на базовый класс и виртуальных методов. Абсолютно все переменные в JavaScript являются объектами (object), а объекты могут хранить ссылку на абсолютно любой дочерний тип данных. Upcast - это называется приведение дочернего значения к базовому типу. Downcast - неявно выполняет виртульные методы.
